## What is this?

Idiomatic, misuse-resistant bindings for [X25519-Cortex-M4][x25519-cortex-m4] ECDH with added EdDSA verification implementation.

[x25519-cortex-m4]: https://github.com/Emill/X25519-Cortex-M4
